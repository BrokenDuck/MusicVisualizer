Project can be viewed at: https://moonlit-kleicha-260704.netlify.app/ (Epilepsy warning)

Project build
-------------

Run: `npm i`
Run: `npm run dev`

Project layout
--------------

The code is in the src folder
All the glsl shaders are writen by Jonathan Maillefaud
All the code is in the main.js file

Ressources used
--------------

Three.js: used for the javascript interface, to create the geometries and to load in the shaders
Tweakpane: used to easily modify values for testing purposes
Vite: used as a framework for development and for glsl integration

Fraction Brownian Motion by Patricio Gonzalez Vivo
COLOR_RAMP macro by Arya Ross
SoftGlitch pass by Arya Ross

Hot Pepper by Alex-Productions | https://onsound.eu/
